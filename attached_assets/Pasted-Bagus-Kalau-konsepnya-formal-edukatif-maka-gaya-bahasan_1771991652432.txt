```

Sistem ini mendukung:
✔ Asesmen Level Kapasitas PJBU
✔ Pengembangan Usaha Berkelanjutan (PUB)
✔ Laporan Kegiatan Usaha Tahunan (LKUT)

Silakan pilih layanan yang Anda butuhkan:
1️⃣ Asesmen Level PJBU
2️⃣ PUB – Pengembangan Usaha Berkelanjutan
3️⃣ LKUT – Laporan Tahunan BUJK
4️⃣ Hubungi Admin

# 🟦 MODUL 1 – ASESMEN LEVEL PJBU (Formal Edukatif)

### Pembuka:

```
Asesmen ini bertujuan untuk:
• Mengukur tingkat kapasitas PJBU
• Mengidentifikasi area pembinaan
• Mendukung peningkatan daya saing BUJK

Silakan pilih jenis usaha:
1️⃣ PJBU-Kontraktor
2️⃣ PJBU-Konsultan
```

---

### Setelah selesai asesmen (contoh output formal edukatif):

```
HASIL ASESMEN PJBU

Skor Total: {{total}} / 100
Level Kapasitas: {{level}}

Interpretasi:
Level ini mencerminkan tingkat kesiapan manajerial dan teknis PJBU dalam mengelola usaha jasa konstruksi.

Rekomendasi Pembinaan Prioritas:
1. {{prio1}}
2. {{prio2}}
3. {{prio3}}

Asosiasi merekomendasikan mengikuti program pembinaan untuk peningkatan level usaha.
```

(Pengembangan Usaha Berkelanjutan)

```

Silakan lengkapi data berikut untuk pendaftaran kegiatan PUB.
