Informasi Umum
Base URL

Production: https://data.inaproc.id
Staging: https://data.eproc.dev
Local: http://localhost:8080
Format Response
Semua response API menggunakan format JSON dengan struktur standar:


{
  "success": boolean,
  "message": "string",
  "data": object | array,
  "meta": {
    "timestamp": "string",
    "version": "string"
  }
}
Authentication
API endpoint memerlukan autentikasi menggunakan salah satu metode berikut:

API Key:


X-API-Key: YOUR_API_KEY
Bearer Token:


Authorization: Bearer YOUR_JWT_TOKEN
Rate Limiting
Endpoint infrastruktur: Tidak ada batasan
Endpoint query: Sesuai dengan kapasitas data source
Endpoint legacy: 100 requests per minute
