## 4) Kapan notifikasi dikirim (biar tidak spam)

Saya sarankan:

### Mode Internal (Anggota Asosiasi)

✅ **SELALU kirim** ke Email + WA admin setelah hasil keluar.

### Mode Publik (Belum Anggota)

Kirim ke admin **hanya kalau**:

* `total >= 65` (lead bagus), ATAU
* user klik “Saya ingin bergabung / minta dihubungi”

Kalau tidak, cukup tampilkan hasil ke user tanpa notifikasi admin.

## 5) Prioritas Pembinaan (rule-based, cocok untuk Gustafta)

Karena kita nggak pakai ranking otomatis, pakai trigger sederhana:

### Kontraktor – contoh rules

* Jika `K_waktu <= 10` → set `prio1="Schedule Control"`
* Jika `K_mutu <= 7` → set `prio2="QA/QC & SOP Mutu"`
* Jika `K_keuangan <= 10` → set `prio3="Cashflow & Laporan Keuangan"`
  (atau aturan lain sesuai skor terendah)

### Konsultan – contoh rules

* Jika `S_waktu_doc <= 14` → “Manajemen Deliverable & Deadline”
* Jika `S_kualitas_doc <= 10` → “Quality Review Dokumen”
* Jika `S_tenaga <= 10` → “Pemenuhan & Manajemen Tenaga Ahli”
