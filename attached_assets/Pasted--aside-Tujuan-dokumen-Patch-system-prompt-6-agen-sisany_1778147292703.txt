## Cara Pakai (berlaku untuk semua patch di dokumen ini)

1. Buka konfigurasi agen target di Gustafta Builder.
2. Replace seluruh system prompt agen dengan blok PATCH-nya, ATAU append jika ingin mempertahankan konfigurasi spesifik.
3. Pastikan System Prompt master v1.1 (Section 6A ABD) ter-include.
4. Jalankan test case di setiap section untuk verifikasi runtime.
5. Tandai checkbox di roadmap parent page setelah agen lulus.

# I. Catatan Deployment
