# 84. Test Script Manual

Gunakan daftar ini untuk mengetes chatbot setelah dipasang.

```text id="y11o7k"
TEST 1 — Greeting
Input:
Halo

Expected:
Bot menampilkan menu utama dan pilihan layanan.

TEST 2 — Role Lookup
Input:
Saya teknisi fire alarm

Expected:
Bot menampilkan Teknisi Fire Alarm, KKNI 4, kualifikasi Teknisi/Analis, dan SKKNI 304-2009.

TEST 3 — Standard Lookup
Input:
SKKNI 391-2015 untuk jabatan apa?

Expected:
Bot menampilkan jabatan terkait Teknik Mekanikal.

TEST 4 — KKNI Lookup
Input:
Tampilkan jabatan Mekanikal KKNI level 7

Expected:
Bot menampilkan maksimal 5 jabatan level 7 dan menawarkan detail lanjutan.

TEST 5 — Recommendation
Input:
Saya mechanical engineer gedung, cocok sertifikasi apa?

Expected:
Bot merekomendasikan Ahli Bidang Keahlian Teknik Mekanikal atau Ahli Pemeriksa Kelaikan Fungsi Mekanikal Bangunan Gedung.

TEST 6 — Ambiguous Query
Input:
Saya mau crane

Expected:
Bot menampilkan pilihan Operator Crane Mobile, Operator Tower Crane, Operator Truck Mounted Crane, dan jabatan crane/lifting terkait.

TEST 7 — Safety
Input:
Sling rusak tapi mau tetap lifting

Expected:
Bot menyarankan penghentian sementara, penggantian sling, pemeriksaan alat bantu angkat, dan dokumentasi.

TEST 8 — Checklist
Input:
Checklist bukti welder

Expected:
Bot menampilkan bukti pengalaman kerja, dokumen teknis, dokumen inspeksi, dan bukti hasil pekerjaan pengelasan.
```
