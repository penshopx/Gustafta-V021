# ✅ Cara Membuatnya Tetap Relevan di Platform Generik

Ubah konsepnya menjadi:

## 🔹 Generic Feature: Advanced Insight / Deep Analysis Layer

Struktur generiknya:

1. Basic Output Layer
2. Advanced Insight Layer (Optional)
3. Strategic Recommendation Layer (Optional)
4. CTA Layer (Conditional)

Ini bisa dipakai untuk:

* Chatbot hukum
* Chatbot pajak
* Chatbot HR
* Chatbot compliance
* Chatbot medis
* Chatbot pendidikan

Semua bisa punya:
“Generate Advanced Insight”

Tanpa menyebut Tender.

---

# 🏗 Bagaimana Implementasinya di Gustafta?

Tambahkan di Orkestrator:

Generic Conditional Output Block:

IF result_score ≥ threshold
→ show Optional Advanced Insight Button

Label default (customizable per chatbot):
“Generate Executive Summary”
atau
“Generate Detailed Insight”
atau
“Deep Analysis”
atau
“Strategic View”

Nama bisa dikustom di level chatbot.

# 🔥 Jadi Sekarang Kita Pisahkan 2 Hal

## 1️⃣ Level Platform (Gustafta Builder)

* Multi-layer output capability
* Conditional layer trigger
* Optional deep insight generator
* Customizable CTA logic

Semua generik.

Di dalam chatbot Tender:

Advanced Insight Layer kita isi dengan:
Executive Compliance Summary.

Tapi di platform, itu cuma:
Layer 2.
