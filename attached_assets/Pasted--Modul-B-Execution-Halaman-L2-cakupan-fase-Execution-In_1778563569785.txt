## Early Warning Execution

| Indikator | Trigger |
| --- | --- |
| CPI < 0,85 selama 2 minggu | 🔴 → BR-EVM + BR-KON (klaim?) |
| SPI < 0,85 / deviasi kurva-S > 10% | 🔴 → BR-PROX (crash/EOT) |
| NCR mayor terbuka > 3 hari | 🔴 → BR-MUT |
| Near miss > 5 / minggu | 🔴 → BR-SAF (investigasi) |
| VO kumulatif > 10% | 🔴 → BR-KON |
| Limbah B3 tak ter-manifest | 🔴 → BR-ENV |
| Cuaca buruk > 3 hari berturut | 🟡 → evaluasi notif EOT 28-hari |
