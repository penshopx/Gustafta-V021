# 🔷 IMPLEMENTASI KE CHATBOT

Flow sederhana:

1. Tentukan jenis usaha (Kontraktor / Konsultan)
2. Sistem akan masuk ke matriks berbeda
3. User menjawab 20–25 pertanyaan
4. Sistem menjumlahkan skor otomatis
5. Output:

```
Hasil Asesmen PJBU:

Skor: 76/100
Level: MADYA

Rekomendasi Upgrade:
- Perkuat sistem SMM
- Tingkatkan pelatihan SDM
- Gunakan software kontrol proyek
```
