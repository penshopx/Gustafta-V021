## 🏗️ Sprint 1 — Core (ORCH + MAPPER + QUALIFY + DOCS + COMPLY)

### Step 1.1 — Buat Orchestrator

1. **Builder → New Agent → Type: OpenClaw Root**
2. Nama: `SBUCLAW-ORCHESTRATOR`
3. **Copy** system prompt dari halaman *System Prompt OpenClaw — SBUClaw* (blok ORCHESTRATOR)
4. **Copy** persona dari halaman *Persona Gustafta* (Identity/Welcome/Greeting/QR)
5. Aktifkan `session_state` dengan schema YAML
6. Aktifkan `callAgentInternal` enabled = `true`
7. **Save & Publish (Draft)**

### Step 1.2 — Buat 4 Agen Inti

Ulangi pola berikut untuk **MAPPER, QUALIFY, DOCS, COMPLY**:

1. **Builder → New Agent → Type: MultiClaw Worker**
2. Nama: `AGENT-{NAME}`
3. Copy system prompt + persona
4. **Bind KB** sesuai *KB INJECT* di system prompt
5. Set `parent_orchestrator = SBUCLAW-ORCHESTRATOR`
6. Set `output_contract = ABD7_STANDARD`
7. Save & Publish (Draft)

### Step 1.3 — Wire Routing

1. Buka SBUCLAW-ORCHESTRATOR → tab **Routing**
2. Tambahkan 4 intent: `scope_and_readiness`, `smart_mapping`, `qualify_check`, `checklist_docs`, `regulasi`
3. Mapping intent → primer + paralel sesuai tabel di system prompt orchestrator
4. Save & Publish

## 🔄 Rollback Plan

Untuk tiap sprint, snapshot diambil **sebelum** publish ke production.

| Skenario | Trigger | Aksi |
| --- | --- | --- |
| Smoke test < 80% lulus | Per sprint | Revert ke snapshot sprint sebelumnya |
| User complaint > 3 dalam 24 jam | Live | Rollback ke last-good build + buka issue |
| INTEGRITY false-positive > 5% | Live | Reduce overlay aggressiveness, retrain lexicon |
| Latensi > 8 detik | Live | Disable agen paralel non-kritis, monitor |

## 🧰 Lampiran: Cheat Sheet `callAgentInternal`

```jsx
// Contoh fan-out di ORCH
const plan = [
  callAgentInternal("MAPPER",   "smart_mapping",   { project_keywords, portfolio_history }),
  callAgentInternal("QUALIFY",  "qualify_check",   { kekayaan_bersih, pengalaman_proyek }),
  callAgentInternal("DOCS",     "checklist_docs",  { use_case, kualifikasi_target }),
  callAgentInternal("SKKMATCH", "skk_align",       { personel }),
];
const results = await Promise.all(plan);
return synthesize(results); // ringkas 1 respons + QR + ABD-7
```

**Catatan**: pastikan timeout per agen ≤ 6 detik; jika satu agen timeout, lanjutkan dengan hasil parsial + `[ASUMSI: agen X tidak merespons; lanjutkan dengan data tersedia]`.
