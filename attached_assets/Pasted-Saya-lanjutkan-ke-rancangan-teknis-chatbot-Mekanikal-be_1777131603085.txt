Saya lanjutkan ke **rancangan teknis chatbot Mekanikal berbasis SKK/SKKNI**. Data acuannya adalah klasifikasi Mekanikal yang memuat jabatan kerja, kualifikasi, jenjang KKNI, standar kompetensi kerja, dan link SKK/SKKNI. 

# 10. Arsitektur chatbot

## A. Komponen utama

```text
User
 ↓
Chatbot Interface
Website / WhatsApp / Telegram / Custom GPT
 ↓
Intent Detector
Mendeteksi maksud pertanyaan user
 ↓
Knowledge Base SKK Mekanikal
Database jabatan kerja, KKNI, SKKNI, link dokumen
 ↓
Response Generator
Menjawab dengan format terstruktur
```

## B. Alur kerja chatbot

```text
1. User bertanya:
   "Saya mau sertifikasi operator crane"

2. Chatbot membaca kata kunci:
   operator, crane

3. Chatbot mencari data terkait:
   - Operator Crane Mobile
   - Operator Tower Crane
   - Operator Truck Mounted Crane
   - Operator Crawler Crane
   - Operator Rough Terrain Crane

4. Chatbot menampilkan pilihan.

5. Jika terlalu banyak pilihan, chatbot bertanya:
   "Jenis crane yang Anda maksud mobile crane, tower crane, atau truck mounted crane?"
```

# 14. Logika pencarian chatbot

Gunakan logika berikut:

```text
1. Ubah input user menjadi huruf kecil.
2. Cocokkan dengan:
   - jabatan_kerja
   - jenjang_jabatan
   - subklasifikasi
   - kata_kunci
   - standar_kompetensi
3. Jika hasil 1 data:
   tampilkan detail.
4. Jika hasil lebih dari 1:
   tampilkan daftar pilihan.
5. Jika tidak ada hasil:
   tampilkan fallback dan saran kata kunci.
```

# 15. Contoh kode sederhana chatbot

Berikut contoh prototype menggunakan JavaScript.

```javascript
const dataMekanikal = [
  {
    jabatan_kerja: "Teknisi Fire Alarm",
    subklasifikasi: "Proteksi Kebakaran",
    jenjang_jabatan: "Teknisi Fire Alarm",
    kualifikasi: "Teknisi/Analis",
    jenjang_kkni: 4,
    standar_kompetensi: "SKKNI 304-2009",
    link_skk: "https://skkni-api.kemnaker.go.id/v1/public/documents/bbd818ec-027a-47ec-8c14-1970313917c0/download",
    kata_kunci: ["fire alarm", "alarm kebakaran", "proteksi kebakaran"]
  },
  {
    jabatan_kerja: "Mekanik Heating, Ventilation, dan Air Condition (HVAC)",
    subklasifikasi: "Teknik Tata Udara dan Refrigasi",
    jenjang_jabatan: "Mekanik HVAC",
    kualifikasi: "Operator",
    jenjang_kkni: 3,
    standar_kompetensi: "SKKNI 298-2009",
    link_skk: "https://skkni-api.kemnaker.go.id/v1/public/documents/d0ad1709-830a-4dee-b2ce-19d94cf92342/download",
    kata_kunci: ["hvac", "ac", "tata udara", "refrigasi"]
  }
];

function cariSKK(input) {
  const query = input.toLowerCase();

  const hasil = dataMekanikal.filter(item => {
    const teksGabungan = [
      item.jabatan_kerja,
      item.subklasifikasi,
      item.jenjang_jabatan,
      item.kualifikasi,
      item.standar_kompetensi,
      ...item.kata_kunci
    ].join(" ").toLowerCase();

    return teksGabungan.includes(query);
  });

  if (hasil.length === 0) {
    return `
Maaf, data belum ditemukan.

Coba gunakan kata kunci seperti:
- HVAC
- plumbing
- tukang las
- crane
- fire alarm
- scaffolding
- alat berat
`;
  }

  return hasil.map(item => `
Jabatan: ${item.jabatan_kerja}
Subklasifikasi: ${item.subklasifikasi}
Jenjang: ${item.jenjang_jabatan}
Kualifikasi: ${item.kualifikasi}
Jenjang KKNI: ${item.jenjang_kkni}
Standar Kompetensi: ${item.standar_kompetensi}
Link SKK/SKKNI: ${item.link_skk || "-"}
`).join("\n---\n");
}

console.log(cariSKK("fire alarm"));
```

# 18. Prompt untuk chatbot versi Custom GPT

Ini prompt yang lebih siap pakai:

```text
Anda adalah MekaSKK Bot, chatbot bidang Mekanikal berbasis SKK/SKKNI.

Anda hanya boleh menjawab berdasarkan database klasifikasi Mekanikal yang diberikan.

Database berisi:
- Subklasifikasi
- Jabatan kerja
- Jenjang jabatan kerja
- Kualifikasi
- Jenjang KKNI
- Standar kompetensi kerja
- Link dokumen SKK/SKKNI

Aturan:
1. Jangan membuat data baru di luar database.
2. Jika data tidak ditemukan, katakan bahwa data belum ditemukan.
3. Jika link SKK tidak tersedia, tulis "Link belum tersedia".
4. Jika standar masih dalam proses penyusunan, sampaikan sesuai data.
5. Jika user menyebut istilah umum seperti "crane", "alat berat", "las", atau "plumbing", tampilkan beberapa pilihan jabatan yang relevan.
6. Jika user meminta rekomendasi, gunakan pengalaman kerja, bidang, dan level untuk memilih jabatan paling sesuai.
7. Gunakan bahasa Indonesia.
8. Jawaban harus ringkas, jelas, dan berbentuk poin.

Format jawaban:
Jabatan:
Subklasifikasi:
Jenjang Jabatan:
Kualifikasi:
Jenjang KKNI:
Standar Kompetensi:
Link SKK/SKKNI:
Catatan:
```
