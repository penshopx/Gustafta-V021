## Test Battery (5 input per agen)

Untuk setiap agen, jalankan 5 input uji:

1. **Input lengkap** вАФ cek output ABD-7 lengkap
2. **Input minim** вАФ cek [ASUMSI] muncul + confidence rendah
3. **Input lintas-domain** вАФ cek callAgentInternal
4. **Input dengan event regulasi** вАФ cek sitasi tepat
5. **Input tanpa pertanyaan eksplisit** вАФ cek ABD-7 follow-up muncul

## Hotfix Library (jika gagal audit)

### Bila ABD-3 Confidence tidak muncul

```
PATCH: Tambah di System Prompt Section 4 (Output):
"WAJIB akhiri SETIAP jawaban substantif dengan: 
Confidence: NN% (0-100)"
```

### Bila ABD-5 Sitasi tidak ada

```
PATCH: Tambah di System Prompt Section sitasi:
"Setiap jawaban substantif HARUS punya minimal 1 sitasi. 
Bila tidak ada sitasi yang sesuai, jawab: 
'Tidak ada referensi regulasi langsung; berdasarkan praktik umum...' 
DAN tambah confidence вЙ§ 50%."
```

### Bila ABD-6 Format ABD-7 tidak konsisten

```
PATCH: Pindahkan Section 4 Format ke awal System Prompt + 
beri contoh konkret 1 output ABD-7 lengkap (few-shot).
```
