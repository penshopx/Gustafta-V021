Lanjut ke **tahap implementasi teknis chatbot SKK Bidang Sipil**: struktur database, logika pencarian, format API, dan template respons. Basis datanya tetap dari file SKK Sipil yang Anda unggah, yang memuat subklasifikasi, jabatan kerja, jenjang jabatan, kualifikasi, KKNI, standar kompetensi, dan link SKK. 

# 1. Arsitektur Chatbot SKK Sipil

```text
USER
↓
WhatsApp / Web Chat / Telegram / GPT Interface
↓
Intent Detector
↓
Entity Extractor
↓
SKK Search Engine
↓
Recommendation Engine
↓
Response Generator
↓
USER
```

## Komponen Utama

```text
1. Intent Detector
   Mendeteksi maksud pengguna:
   - Cari SKK
   - Rekomendasi SKK
   - Cek KKNI
   - Ambil link SKKNI
   - Bandingkan jabatan
   - Cari berdasarkan subklasifikasi

2. Entity Extractor
   Mengambil kata penting:
   - Gedung
   - Jalan
   - Jembatan
   - Geoteknik
   - Ahli
   - Pelaksana
   - Pengawas
   - Freshgraduate
   - KKNI 7
   - SKKNI 126-2021

3. SKK Search Engine
   Mencari data paling cocok di database.

4. Recommendation Engine
   Memberi rekomendasi berdasarkan:
   - bidang
   - pengalaman
   - posisi
   - jenis proyek
   - target level

5. Response Generator
   Menjawab dengan format rapi dan mudah dipahami.
```

# 3. Status Dokumen

Gunakan status agar chatbot tahu apakah link bisa diberikan atau belum.

```text
available
Link dokumen tersedia.

not_available
Link tidak tersedia atau tertulis "-".

draft
Masih RSKKNI / akan ditetapkan.

lpjk_scheme
Masih SKEMA LPJK / akan disusun SKKNI/SKK Khusus.

skk_khusus
Menggunakan SKK Khusus, bukan SKKNI umum.
```

Contoh respons bila link tidak tersedia:

```text
Untuk jabatan ini, dokumen SKKNI belum tersedia pada database.
Status saat ini: SKEMA LPJK / Akan disusun SKKNI / SKK Khusus.
Silakan cek kembali ke LPJK/LSP terkait untuk informasi terbaru.
```

# 5. Logika Pencarian Chatbot

## A. Pencarian Berdasarkan Jabatan

User:

```text
Cari Ahli Teknik Jalan
```

Query:

```sql
SELECT *
FROM skk_sipil
WHERE jabatan_kerja LIKE '%Ahli Teknik Jalan%'
   OR jenjang_jabatan LIKE '%Ahli Teknik Jalan%'
   OR kata_kunci LIKE '%ahli teknik jalan%';
```

Jawaban:

```text
Ditemukan SKK untuk Ahli Teknik Jalan:

1. Ahli Muda Teknik Jalan
   - Subklasifikasi: Jalan
   - Kualifikasi: Ahli
   - KKNI: 7
   - Standar: SKKNI 126-2021

2. Ahli Madya Teknik Jalan
   - Subklasifikasi: Jalan
   - Kualifikasi: Ahli
   - KKNI: 8
   - Standar: SKKNI 126-2021

3. Ahli Utama Teknik Jalan
   - Subklasifikasi: Jalan
   - Kualifikasi: Ahli
   - KKNI: 9
   - Standar: SKKNI 126-2021
```

## B. Pencarian Berdasarkan Subklasifikasi

```sql
SELECT *
```

```text
Berikut daftar SKK subklasifikasi Jembatan:

- Pengawas Lapangan Pekerjaan Jembatan
- Mandor Pemasangan Rangka Baja Jembatan
```

# 6. Logika Rekomendasi Berdasarkan Pengalaman

## Aturan Dasar

```text
0 tahun / fresh graduate
→ Freshgraduate atau Ahli Muda Freshgraduate jika tersedia.

1–3 tahun
→ Operator Level 1–3.

4–6 tahun
→ Teknisi/Analis KKNI 4–6.

7–10 tahun
→ Ahli Muda KKNI 7 atau Ahli Madya KKNI 8.

>10 tahun
→ Ahli Madya KKNI 8 atau Ahli Utama KKNI 9.
```

## Contoh Pseudocode

```python
def rekomendasi_level(pengalaman_tahun, target=None):
    if pengalaman_tahun == 0:
        return ["Freshgraduate", "Ahli Muda"]
    
    if pengalaman_tahun <= 3:
        return ["Operator"]
    
    if pengalaman_tahun <= 6:
        return ["Teknisi/Analis"]
    
    if pengalaman_tahun <= 10:
        return ["Ahli Muda", "Ahli Madya"]
    
    return ["Ahli Madya", "Ahli Utama"]
```

# 7. Contoh Rekomendasi Otomatis

User:

```text
Saya pengalaman 5 tahun di proyek jembatan sebagai pelaksana lapangan.
```

Chatbot:

```text
Berdasarkan profil Anda:

Bidang utama:
Jembatan

Posisi:
Pelaksana lapangan

Pengalaman:
5 tahun

Target level yang cocok:
Teknisi/Analis, sekitar KKNI 4–6.

Rekomendasi utama:

Pelaksana Pemeliharaan Jembatan
- Subklasifikasi: Jembatan
- Kualifikasi: Teknisi/Analis
- Jenjang KKNI: 4–6
- Standar: SKKNI 195-2015

Alternatif:
1. Teknisi Jembatan Rangka Baja
   Cocok jika pengalaman Anda banyak pada struktur jembatan rangka baja.

2. Pengawas Lapangan Pekerjaan Jembatan
   Cocok jika tanggung jawab Anda sudah lebih banyak pada pengawasan pekerjaan.

Catatan:
Rekomendasi ini bersifat awal. Penetapan final tetap mengikuti asesmen LSP/LPJK dan bukti pengalaman kerja.
```

# 8. Endpoint API untuk Chatbot

Untuk sistem berbasis web atau WhatsApp bot, siapkan endpoint berikut.

## A. Search SKK

```http
GET /api/skk/search?q=ahli teknik jalan
```

Response:

```json
{
  "query": "ahli teknik jalan",
  "results": [
    {
      "subklasifikasi": "Jalan",
      "jabatan_kerja": "Ahli Teknik Jalan",
      "jenjang_jabatan": "Ahli Muda Teknik Jalan",
      "kualifikasi": "Ahli",
      "kkni": 7,
      "standar": "SKKNI 126-2021",
      "link": "https://skkni-api.kemnaker.go.id/..."
    }
  ]
}
```

## B. Rekomendasi SKK

```http
POST /api/skk/recommend
```

Body:

```json
{
  "bidang": "Gedung",
  "posisi": "Pelaksana Lapangan",
  "pengalaman_tahun": 6,
  "jenis_proyek": "Gedung bertingkat",
  "target_level": "Teknisi/Analis"
}
```

Response:

```json
{
  "rekomendasi_utama": {
    "jabatan": "Pelaksana Lapangan Pekerjaan Gedung Madya",
    "subklasifikasi": "Gedung",
    "kualifikasi": "Teknisi/Analis",
    "kkni": 5,
    "standar": "SKKNI 193-2021"
  },
  "alternatif": [
    "Pelaksana Lapangan Pekerjaan Gedung Muda",
    "Pengawas Pekerjaan Struktur Bangunan Gedung"
  ],
  "catatan": "Rekomendasi awal. Keputusan final mengikuti asesmen LSP/LPJK."
}
```

# 9. Template Prompt untuk Model AI + Database

Gunakan prompt ini untuk sistem RAG/chatbot:

```text
Anda adalah Asisten SKK Bidang Sipil.

Anda hanya boleh menjawab berdasarkan database SKK Sipil yang tersedia.

Database berisi:
- klasifikasi
- subklasifikasi
- jabatan kerja
- jenjang jabatan kerja
- kualifikasi
- jenjang KKNI
- standar kompetensi kerja
- link SKK

Tugas Anda:
1. Mencari SKK berdasarkan jabatan, subklasifikasi, KKNI, atau standar kompetensi.
2. Menampilkan semua jenjang bila satu jabatan memiliki beberapa level.
3. Memberikan rekomendasi SKK berdasarkan profil pengguna.
4. Menjelaskan alasan kecocokan secara praktis.
5. Memberikan link dokumen jika tersedia.
6. Menjelaskan status dokumen jika link tidak tersedia.
7. Tidak mengarang data di luar database.
8. Tidak menjamin kelulusan sertifikasi.

Jika data pengguna belum lengkap, tanyakan maksimal 3 pertanyaan:
- Bidang proyek utama?
- Posisi saat ini?
- Lama pengalaman kerja?
```

# 13. Fitur Lanjutan: “Cek Kecocokan Proyek”

```text
Untuk proyek jalan nasional, personel apa saja yang sebaiknya punya SKK?
```

Chatbot:

```text
Untuk proyek jalan, SKK yang relevan dapat mencakup:

1. Ahli Teknik Jalan
   Untuk perencanaan, evaluasi teknis, dan pengambilan keputusan teknis.

2. Manajer Pelaksanaan Pekerjaan Jalan/Jembatan
   Untuk pengelolaan pelaksanaan proyek jalan/jembatan.

3. Pelaksana Lapangan Pekerjaan Jalan
   Untuk pelaksanaan teknis lapangan.

4. Pengawas Lapangan Pekerjaan Jalan
   Untuk pengendalian mutu, metode, dan kepatuhan pelaksanaan.

5. Ahli Material Jalan
   Untuk pengendalian material perkerasan dan mutu material.

Catatan:
Kebutuhan personel final harus mengikuti dokumen tender, kontrak, regulasi, dan ketentuan sertifikasi yang berlaku.
```
