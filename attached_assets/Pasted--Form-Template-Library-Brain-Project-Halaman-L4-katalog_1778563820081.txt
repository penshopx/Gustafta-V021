# 📝 Form & Template Library — Brain Project

> Halaman L4 — katalog form & template siap-pakai. Untuk implementasi: setiap template di-create sebagai *database template page* di database operasional Brain Project (LHP / NCR / K3 / Risiko / EVM).
> 

## Cara Pakai

1. Buka database operasional target (mis. [](https://www.notion.so/d84913f1c9b449a19a2241ff2dca523c?pvs=21)).
2. Klik *New → dropdown template → pilih template* yang diinginkan.
3. Atau mention `@Brain Project` → "Tolong susun [template name] untuk pekerjaan X".
