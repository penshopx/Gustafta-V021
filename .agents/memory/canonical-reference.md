# Acuan kanonik proyek

Sumber kebenaran identitas, visi, dan arsitektur Gustafta adalah **`replit.md`** di root repo.

**Why:** Dahulu ada dokumen genesis terpisah `.local/gustafta-pedoman.md`. Dokumen itu
sudah **TIDAK ADA** lagi. Mencarinya hanya membuang waktu dan bisa menyesatkan.

**How to apply:** Saat butuh konteks produk/arsitektur tingkat tinggi, baca `replit.md`
(bagian "Product", "Architecture decisions", "Where things live"). Jangan cari pedoman lain.
